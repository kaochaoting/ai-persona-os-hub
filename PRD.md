# Hub — Hermes Agent 社群中樞

## 產品需求文檔 (PRD)
> 版本 1.0 · 2026-05-11

---

## 1. 產品概述

### 1.1 產品願景
成為 Hermes Agent 生態系的社群中心 — 一個讓使用者交流技能、展示 AI Agent、分享經驗、取得教學資源的開放平台。

### 1.2 目標用戶
- Hermes Agent 的使用者（技術背景、AI 愛好者）
- OpenClaw 社群成員
- 想建立自己 AI Agent 人格的一般用戶
- 潛在的 AI 開發者與研究者

### 1.3 核心價值
- **開放**：所有資源免費使用，MIT 授權
- **實用**：每個功能解決一個真實問題
- **簡潔**：無印良品風格的純粹設計，不被多餘元素干擾

## 5. 設計原則

### 5.1 無印良品風格
- **空白是裝飾**：大量留白，讓內容呼吸
- **不標榜品牌**：不強調「我們」，專注呈現內容
- **功能即美學**：不需要的裝飾全部刪除
- **自然質感**：暖色調中性色，避免冷色工業感
- **一致樸素**：所有元素統一風格，不突出任何單一元素

### 5.2 色彩系統
```css
/* 靈感：紙、木頭、墨水 */
背景:       #FAFAF7  /* 暖白紙張色 */
卡片:       #FFFFFF  /* 純白 */
主文字:     #2C2C2C  /* 濃墨 */
次要文字:   #888888  /* 淡墨 */
強調色:     #8B7355  /* 暖棕 / 木質 */
分隔線:     #E8E4DE  /* 極淡灰 */
懸停:       #F0EDE6  /* 極淡暖灰 */
```

### 5.3 排版
- 字體：系統無襯線（-apple-system, sans-serif）
- 行高：1.8（文章）、1.4（標題）
- 最大閱讀寬度：720px（文章）、1200px（目錄）
- 不使用粗體之外的裝飾字型
