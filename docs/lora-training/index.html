SDXL LoRA 訓練教學

8GB VRAM 上的角色訓練

環境

  • GPU：NVIDIA GTX 1070Ti（8GB VRAM）
  • 框架：kohya_ss / sd-scripts
  • PyTorch 2.6.0+cu124
cd ~/kohya_ss
source venv/bin/activate
accelerate launch sd-scripts/sdxl_train_network.py \
  --pretrained_model_name_or_path=model.safetensors \
  --train_data_dir=dataset \
  --output_dir=output \
  --network_dim=16 \
  --learning_rate=1e-4 \
  --max_train_steps=2000

資料集

資料夾命名：{重複數}_{標記}，例如 10_xiaowen

關鍵參數

  • network_dim=16 — 8GB 最佳 LoRA 維度
  • max_train_steps=2000 — 角色訓練標準步數
  • 學習率 1e-4 — AdamW 起點

注意事項

  • HF_HUB_OFFLINE=1 — 跳過 HuggingFace 檢查
  • WANDB_MODE=disabled — 關閉 wandb
  • 勿用 --lowram、--no_half_vae
