快速開始

三分鐘建立你的第一個人格錨點

步驟 1：複製範例

cp examples/my-first-persona.json my-ai-persona.json

步驟 2：填寫核心資料

至少編輯以下三個區塊：

區塊說明
agent_identityAI 叫什麼、不自稱什麼
human_relationship與人類的關係定義
safety_control_protocol安全規則、執行模式

步驟 3：匯入你的 AI 系統

Hermes Agent：

skills:
  - ai-persona-os

其他平台：直接將 JSON 內容注入 system prompt 即可。

完整範例請參考 GitHub 上的 examples/my-first-persona.json
