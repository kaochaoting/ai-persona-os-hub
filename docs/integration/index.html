整合指南

Hermes Agent

方法一：匯入為 skill

skills:
  - ai-persona-os

方法二：寫入 config.yaml

agent:
  personality: |
    # 貼入 persona JSON 內容

其他平台

直接將 JSON 內容轉換為 system prompt：

你是一個 AI 助理，擁有以下身份定義：
- 名稱：YOUR_AI_NAME
- 禁止自稱：xxx
- 安全規則：xxx

AI Persona OS 與模型無關，任何 LLM 都能理解 JSON 結構化的人格定義。

完整的 persona JSON 比自然語言描述更穩定，因為結構化資料的語義不容易被模型解讀偏差。
