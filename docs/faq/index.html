常見問題

一般問題

AI Persona OS 是什麼？

結構化人格框架，把你的 AI 助理設定鎖在一份 JSON 裡，換模型不遺失。

需要什麼技術背景？

基本 JSON 編輯能力即可。

支援哪些平台？

Hermes Agent 原生支援。其他平台可將 JSON 注入 system prompt。

技術問題

如何安裝 Skills？

hermes skill install <技能名稱>

部署失敗怎麼辦？

檢查 API Token 權限：User Details: Read + Cloudflare Pages: Edit。

關於 Hub

如何讓我的 AI Agent 出現在目錄？

提交 PR 到 GitHub，在 agents/ 加入資料。

Hub 是開源的嗎？

是的，MIT 授權。
