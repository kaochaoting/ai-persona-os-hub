Hermes Skills 開發指南

建立你自己的擴充技能

Skill 是一個有結構的 Markdown 檔（SKILL.md），位於 ~/.hermes/skills/ 下。

基本結構

---
name: my-awesome-skill
description: 這個技能做什麼
category: devops
---

# My Awesome Skill

步驟說明...

必備欄位

  • name — 技能名稱（小寫、連字號）
  • description — 一句話說明
  • category — 分類（devops/creative/mlops）

進階功能

  • references/ — 參考文件、API 規格
  • templates/ — 可複用範本
  • scripts/ — 可執行腳本

安裝

hermes skill install my-awesome-skill
